# Inventory Management System 

A Java-based Inventory Management System developed to manage product stock, quantity updates, low-stock alerts, and product operations using arrays with proper input validation.

## Features
- Add new products
- Update product quantity
- Search products by ID or name
- Remove products
- Low-stock alerts
- Input validation handling
- Display inventory details

## Technologies Used
- Java
- Arrays
- Scanner Class
- Object-Oriented Programming Basics

## Project Highlights
- Parallel array-based data management
- Real-time stock updates
- Beginner-friendly console application
- Demonstrates Java fundamentals and logic building
